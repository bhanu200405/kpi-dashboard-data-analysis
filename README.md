# 📊 KPI Dashboard Data Analysis Project

## 🧾 Project Overview
This project focuses on analyzing business performance using key performance indicators (KPIs).  
It uses Python, SQL, and data visualization tools to extract insights from data.

---

## 🎯 Objectives
- Analyze sales and profit data
- Identify top customers and products
- Study monthly sales trends
- Detect outliers in pricing
- Generate business insights

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (Data analysis queries)
- Excel / Power BI (Dashboard creation)

---

## 📂 Project Structure
- data → Raw dataset
- notebook → Python analysis
- sql → SQL queries
- dashboards → Final dashboard file
- images → Visual outputs

---

## 📊 Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Top Customers
- Monthly Sales Trend
